# Social Media Analytics & Competitor Intelligence Dashboard

## Overview
A full-stack web dashboard to analyse social media performance and benchmark it against competitors.

## Tech Stack
- Frontend: React, Chart.js
- Backend: FastAPI (Python)
- Data: Script-based mocked social media data
- Database: SQLite (optional extension)

## Features
- Follower growth tracking
- Competitor comparison
- Engagement-based insights
- Explainable AI recommendations

## Note
Real social media APIs were mocked due to platform access restrictions.
